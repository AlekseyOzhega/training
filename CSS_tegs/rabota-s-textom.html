Трансформации

  • color

    Свойство задаёт цвет текста.

    Дополнительная информация

  • font-family

    С помощью font-family можно задать семейство шрифтов, которым будет написан текст. Можно прописать конкретно, например "Arial". А можно задать только желаемый тип: например, любое семейство шрифтов с засечками (serif) или без засечек (sans-serif).

    Дополнительная информация

  • font-size

    Используй это свойство, чтобы задать размер текста.

    Дополнительная информация

  • line-height

    задаёт интервал между строчками текста, который называют интерлиньяж.

    Дополнительная информация

  • text-align

    управляет выравниванием контента (к примеру текста) по горизонтали внутри родительского элемента.

    Дополнительная информация

  • text-transform

    Свойство text-transform позволяет трансформировать буквы в тексте. С помощью этого свойства можно сделать текст из одних заглавных или наоборот, из одних маленьких букв вне зависимости от формата исходного текста.

    Дополнительная информация

  • vertical-align

    Выравниваем элемент по вертикали без флексов и гридов.

    Дополнительная информация

  • @font-face

    С помощью данной функции можно подключить и использовать на странице любой шрифт из файла.

    Дополнительная информация

  • font-style

    Определяет начертание шрифта: обычное, курсивное или наклонное.

    Дополнительная информация

  • font-weight

    Изменяет толщину букв в тексте.

    Дополнительная информация

  • font-display

    — это не CSS-свойство, которое можно применить к конкретному элементу. Это так называемый дескриптор. Он применяется только внутри секции правил @font-face и описывает, как будет отображаться текст в зависимости от того, загружен шрифт или нет.

    Дополнительная информация

  • font

    Одно свойство для всех настроек шрифта.

    Дополнительная информация

  • font-smooth

    Сглаживаем буквы текста на macOS.

    Дополнительная информация

  • letter-spacing

    задаёт межбуквенное расстояние — интервал между символами в тексте.

    Дополнительная информация

  • text-decoration

    позволяет добавить декоративные линии тексту. Текст можно подчеркнуть, перечеркнуть или добавить линию над текстом.

    Дополнительная информация

  • text-decoration-color

    Свойство text-decoration-color определяет цвет декоративной линии (подчёркивания, зачёркивания и других), заданной при помощи text-decoration-line.

    Дополнительная информация

  • text-decoration-line

    Свойство text-decoration-line создаёт декоративные линии в тексте.

    Дополнительная информация

  • text-decoration-style

    Свойство управляет стилем декоративной линии для текста (подчёркивание, перечёркивание и другие).

    Дополнительная информация

  • text-decoration-thickness

    Свойство text-decoration-thickness управляет толщиной декоративной линии у текста, если она задана свойством text-decoration.

    Дополнительная информация

  • text-decoration-skip-ink

    Свойство text-decoration-skip-ink управляет внешним видом верхнего и нижнего подчёркиваний. Если текст нужно перечеркнуть, то свойство text-decoration-skip-ink будет проигнорировано.
