Директивы

  • @font-face

    С помощью этой директивы можно подключить и использовать на странице любой шрифт из файла.

    Дополнительная информация

  • @import

    При помощи директивы @import можно импортировать один файл со стилями в другой файл со стилями.

    Дополнительная информация

  • @keyframes

    Директива @keyframes используется для создания ключевых кадров CSS-анимаций.

    Дополнительная информация

  • @media

    Директива, которая позволяет задавать разные стили для разных параметров экрана — ширины, высоты, ориентации и даже типа устройства.

    Дополнительная информация

  • @supports

    Директива @supports позволяет проверить, поддерживает ли браузер свойство, правило или CSS-селектор. Если условие срабатывает, то будет выполнен CSS-код, написанный внутри фигурных скобок.

    Дополнительная информация

  • @layer

    Директива @layer, дополнительная функция layer() и ключевое слово layer дают вам возможность управлять каскадными слоями.

    Дополнительная информация
