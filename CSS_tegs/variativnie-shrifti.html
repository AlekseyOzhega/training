Трансформации

  • font-feature-settings

    Свойство font-feature-settings позволяет управлять функциями шрифтов с расширенным набором символов.

    Дополнительная информация

  • font-kerning

    Свойство font-kerning позволяет включать или выключать встроенный кернинг шрифта. Кернингом называют расстояние между парами букв. При включённом кернинге буквы приближаются или отдаляются друг от друга, чтобы компенсировать особенности их формы и сохранить горизонтальный ритм текста.

    Дополнительная информация

  • font-language-override

    Свойство font-language-override контролирует использование в шрифте глифов, относящихся к конкретному языку.

    Дополнительная информация

  • font-optical-sizing

    Свойство font-optical-sizing позволяет браузеру менять начертание глифов для улучшения разборчивости текста. Мелкий текст будет отображаться более толстым начертанием, а крупный текст, напротив, может отображаться более тонким начертанием. Работает для шрифтов, в которых предусмотрена ось оптического изменения размера.

    Дополнительная информация

  • font-size-adjust

    Свойство font-size-adjust позволяет задать высоту строчных букв относительно прописных, размер которых задаётся с помощью font-size.

    Дополнительная информация

  • font-stretch

    Свойство font-stretch устанавливает узкое, нормальное или широкое начертание шрифта.

    Дополнительная информация

  • font-synthesis

    Свойство font-synthesis контролирует, какие недостающие начертания (жирный, курсив или капитель) могут быть синтезированы браузером.

    Дополнительная информация

  • font-variant

    Шорткат font-variant даёт возможность задать значения для следующих свойств:
    font-variant-alternates;
    font-variant-caps;
    font-variant-east-asian;
    font-variant-ligatures;
    font-variant-numeric.

    Дополнительная информация
