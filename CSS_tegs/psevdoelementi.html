  • ::placeholder

    Псевдоэлемент ::placeholder используется для стилизации текста-подсказки (плейсхолдера) в полях ввода <input> и <textarea>.

    Дополнительная информация

  • ::first-letter

    Псевдоэлемент ::first-letter позволяет задать стили для первой буквы первой строки блочного элемента, если перед текстом нет другого содержимого.

    Дополнительная информация

  • ::first-line

    Псевдоэлемент ::first-line позволяет задать стили для первой строки текста, расположенного внутри блочного элемента. Текст, попадающий под действие этого псевдоэлемента, вычисляется динамически при изменении ширины блока.

    Дополнительная информация

  • ::selection

    Псевдоэлемент ::selection позволяет применить стили к пользовательскому выделению (например, к выделенному с помощью мыши тексту) и изменить его вид. Это полезно, если вы хотите оформить выделение текста в соответствии с вашим дизайном.

    Дополнительная информация

  • ::marker

    ::marker — псевдоэлемент, отвечающий за маркерное поле. В нём находятся, например, маркеры списка.
