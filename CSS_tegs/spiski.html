  • list-style

    Шорткат list-style позволяет одновременно задать значение для следующих свойств:

    list-style-type;

    list-style-position;

    list-style-image.

    Дополнительная информация

  • list-style-image

    Свойство устанавливает картинку в качестве маркера у списка.

    Дополнительная информация

  • list-style-position

    Свойство list-style-position задаёт положение маркера списка относительно элемента списка.
