Флексбоксы

  • Гайд по flexbox

    Флексбокс — гибкое распределение места между элементами, гибкая расстановка, выравнивание, гибкое управление.

    Дополнительная информация

  • flex-shrink и flex-grow

    — это CSS-свойства, определяющие, какое количество свободного пространства флекс-контейнера может занимать текущий элемент.

    Дополнительная информация

  • flex

    Короткая запись для управления размерами флекс-элемента.

    Дополнительная информация

  • flex-basis

    Свойство flex-basis указывает на размер элемента до того, как свободное место будет распределено (см. flex-grow).

    Дополнительная информация

  • flex-direction

    это свойство расставляет флекс-элементы в ряд или в колонку.

    Дополнительная информация

  • flex-flow

    Шорткат, позволяющий одновременно задать значения свойствам flex-direction и flex-wrap.

    Дополнительная информация

  • flex-grow

    Это свойство указывает, может ли вырастать флекс-элемент при наличии свободного места, и на сколько.

    Дополнительная информация

  • flex-shrink

    Если в контейнере не хватает места для расположения всех элементов без изменения размеров, то свойство flex-shrink указывает, в каких пропорциях элемент будет уменьшаться.

    Дополнительная информация

  • flex-wrap

    Позволяем флекс-элементам переноситься на новые строки.

    Дополнительная информация

  • order

    При помощи свойства order можно менять порядок отображения флекс-элементов внутри флекс-контейнера.
