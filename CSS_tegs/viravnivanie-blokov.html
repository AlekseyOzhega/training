Выравнивание блоков

  • justify-content

    Свойство для выравнивания элементов по основной оси в гридах и флексах.

    Дополнительная информация

  • justify-items

    Свойство, с помощью которого задаётся выравнивание грид-элементов по горизонтальной оси. Применяется ко всем элементам внутри грид-родителя.

    Дополнительная информация

  • justify-self

    С помощью этого свойства можно установить горизонтальное выравнивание для отдельного элемента, отличное от выравнивания, заданного грид-родителю.

    Дополнительная информация

  • align-content

    Свойство распределяет свободное пространство по поперечной оси между строками флекс-элементов.

    Дополнительная информация

  • align-items

    Свойство для выравнивания элементов внутри контейнера по поперечной оси.

    Дополнительная информация

  • align-self

    При помощи этого свойства можно выровнять один или несколько элементов по вертикальной оси иначе, чем задано у родительского элемента.

    Дополнительная информация

  • place-content

    — это свойство-шорткат для align-content и justify-content. С его помощью можно выровнять элементы во флекс- или грид-контейнерах.

    Дополнительная информация

  • place-items

    Шорткат для указания значений сразу и для align-items и для justify-items. Указывать нужно именно в таком порядке.

    Дополнительная информация

  • place-self

    Шорткат для одновременного указания значений для свойств justify-self и align-self.

    Дополнительная информация

  • gap

    Шорткат для записи значений свойств row-gap и column-gap. Значения разделяются пробелом. Работает как с гридами, так и с флексбоксами.

    Дополнительная информация

  • row-gap, column-gap

    Задают отступы между рядами и колонками в грид-раскладке.
