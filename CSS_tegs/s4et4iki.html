Счётчики

  • Счётчики в CSS

    CSS-счётчики — это мощный инструмент для нумерации любых элементов страницы, не только списков.

    Дополнительная информация

  • counter-set

    С помощью свойства counter-set можно указать конкретное числовое значение определённому счётчику.

    Дополнительная информация

  • counter-reset

    С помощью свойства counter-reset создаются новые счётчики на элементе.

    Дополнительная информация

  • counter-increment

    С помощью свойства counter-increment можно указать число, на которое будет увеличиваться значение CSS-счётчика.

    Дополнительная информация

  • counter(), counters()

    Сами по себе счётчики не имеют видимого эффекта, но их значения можно использовать с помощью функций counter() и counters().
