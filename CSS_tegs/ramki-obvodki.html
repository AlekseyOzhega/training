Рамки, обводки и тени

  • border

    Свойство border отвечает за отрисовку видимой границы блока. Часто видимую границу блока называют рамкой.

    Дополнительная информация

  • border-block

    border-block — это логическое свойство-шорткат. Он создаёт у элемента рамки по блочной оси.

    Дополнительная информация

  • border-inline

    border-inline — это логическое свойство-шорткат. Оно создаёт у элемента рамки по строчной оси.

    Дополнительная информация

  • border-collapse

    Свойство border-collapse управляет отображением границ таблицы и её ячеек.

    Дополнительная информация

  • border-image

    Свойство border-image пригодится в том случае, если вы хотите задать блоку необычную рамку, а не просто сплошную или прерывистую линию одного цвета.

    Дополнительная информация

  • border-style

    Свойство border-style управляет стилем рамки элемента.

    Дополнительная информация

  • border-radius

    Свойство border-radius закругляет углы почти у любого элемента. И даже если у блока не задана явная рамка.

    Дополнительная информация

  • border-spacing

    Свойство border-spacing задаёт отступ между рамками ячеек таблицы.

    Дополнительная информация

  • border-color

    Свойство border-color управляет цветом рамки элемента.

    Дополнительная информация

  • border-width

    Свойство border-width управляет толщиной рамки элемента.

    Дополнительная информация

  • outline

    Обводка вокруг элемента, не влияющая на его размеры.

    Дополнительная информация

  • outline-color

    Отдельное свойство для управления цветом обводки элемента.

    Дополнительная информация

  • outline-offset

    Свойство outline-offset устанавливает расстояние между фактическим краем элемента и внутренним краем обводки, заданной при помощи outline.

    Дополнительная информация

  • outline-style

    Свойство outline-style задаёт стиль обводки.
