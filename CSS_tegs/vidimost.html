Видимость

  • opacity

    Свойство opacity управляет прозрачностью элемента.

    Дополнительная информация

  • visibility

    Свойство visibility прячет элемент от глаз пользователя.

    Дополнительная информация

  • clip

    С помощью свойства clip можно обрезать элемент, оставив видимой только его часть.

    Это свойство устарело. Вместо него следует использовать clip-path

    Дополнительная информация

  • clip-path

    С помощью clip-path можно задавать видимую область элемента. Всё, что выходит за её пределы, скрывается.

    Дополнительная информация

  • will-change

    Свойство will-change позволяет заранее сообщить браузеру об изменениях (анимация, перемещение и т.д), которые могут произойти с элементом.

    Дополнительная информация
