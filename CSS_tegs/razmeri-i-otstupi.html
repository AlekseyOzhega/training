Размеры и отступы

  • width

    Свойство width отвечает за ширину элемента.

    Дополнительная информация

  • height

    При помощи свойства height можно задать высоту любому блочному (block) или строчно-блочному (inline-block) элементу.

    Дополнительная информация

  • padding

    войство padding — или внутренний отступ — позволяет оттолкнуть контент от границ родительского элемента.

    Дополнительная информация

  • margin

    Задаёт размер внешнего отступа вокруг элемента. Иногда внешние отступы называют полями.

    Дополнительная информация

  • box-sizing

    При помощи свойства box-sizing можно изменить то, как браузер будет рассчитывать размеры элемента.
