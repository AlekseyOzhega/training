Работа с изображениями

  • object-fit

    Свойство, которое позволяет управлять тем, как картинка <img> или видео <video> будет подстраиваться под заданные размеры.

    Дополнительная информация

  • object-position

    Свойство помогает спозиционировать отрисовку картинки <img> или видео <video> внутри контейнера, если к нему применили свойство object-fit.

    Дополнительная информация

  • aspect-ratio

    Свойство задаёт соотношение сторон для элемента.

    Дополнительная информация

  • filter

    Свойство filter даёт возможность прямо в CSS применять к элементам различные графические эффекты.
