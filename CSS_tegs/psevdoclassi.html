Псевдоклассы

  • :link

    Псевдокласс :link используется для оформления ссылок, которые пользователь ещё никогда не открывал (в рамках текущего домена).

  • :visited

    Псевдокласс :visited добавляется ссылкам, по которым уже переходил пользователь.

    Пример ссылки посещённой

    Дополнительная информация

  • :hover

    Оживляем элементы, реагируем на наведение курсора.

    Случайный текст

    Ссылка

    Кнопка

    Дополнительная информация

  • :active

    Псевдокласс :active позволяет задать стили для элемента, с которым происходит взаимодействие прямо сейчас. Например, можно задать кнопке стиль, который будет виден в тот краткий миг, когда на кнопке зажата клавиша мыши.

    Дополнительная информация

  • :target

    Есть ссылки которые ведут не просто на страницу, а на конкретный HTML-элемент внутри. Псевдокласс :target управляет стилем этого элемента.

    Дополнительная информация

  • Группа child

    При помощи этих псевдоклассов можно удобно выбирать элементы по их порядковому номеру внутри родительского элемента.

    Дополнительная информация

  • Группа type

    Эти псевдоклассы позволяют выбирать среди элементов одного типа по порядковому номеру.

    Дополнительная информация

  • :focus

    Сайтом можно пользоваться не только с использованием мышки, но и при помощи клавиатуры. При переключении клавишей Tab элементу добавляется псевдокласс :focus, и мы можем задать для него красивые стили.

    Дополнительная информация

  • :focus-visible

    Псевдокласс :focus-visible используется, когда действительно нужно визуально выделить элементы в фокусе.

    Дополнительная информация

  • :focus-within

    Псевдокласс :focus-within применяется к элементам, которые либо сами находятся в фокусе, либо имеют дочерние элементы в фокусе.

    Дополнительная информация

  • :not

    Очень полезный псевдокласс, который поможет вам неплохо так сэкономить строчки кода. Суть работы этого парня — отсечь все элементы, не подходящие под условие. Звучит сложнее, чем выглядит.

    Дополнительная информация

  • :checked

    Псевдокласс, который активируется, когда пользователь отмечает чекбокс или выбирает одну из радиокнопок.

    Дополнительная информация

  • :disabled, :enabled

    Псевдоклассы :disabled и :enabled помогают стилизовать интерактивные элементы — на которые можно и нельзя нажать.

    Дополнительная информация

  • :empty

    Псевдокласс :empty используется для выбора пустых элементов. Пустыми считаются элементы без потомков и текста.

    Дополнительная информация

  • :has()

    Функция-псевдокласс :has()позволяет уточнить основной селектор дополнительным. Это единственный способ выбрать элемент на основе дочернего или соседнего элемента посредством CSS.

    Дополнительная информация

  • :is()

    Функция-псевдокласс :is() принимает один или несколько селекторов в качестве аргумента. Браузер при чтении применяет стили к любому из селекторов-аргументов.

    Дополнительная информация

  • :where()

    Функция-псевдокласс :where() принимает один или несколько селекторов в качестве аргумента. Браузер при чтении применяет стили к любому из селекторов-аргументов.

    Дополнительная информация

  • :in-range, :out-of-range

    Правильно ли заполнено поле ввода с цифрами? Проверяем прямо в браузере.

    Дополнительная информация

  • :required

    Стилизуем поля формы, обязательные для заполнения.

    Дополнительная информация

  • :optional

    Показываем, какие поля заполнять необязательно.

    Дополнительная информация

  • :invalid, :valid

    Псевдоклассы используются для стилизации полей формы, филдсетов, либо самой формы:

    • :invalid для заполнений с ошибкой;
    • :valid для верных заполнений.
    Дополнительная информация

  • :placeholder-shown

    Псевдокласс :placeholder-shown выбирает на странице все <input> или <textarea>, у которых показывается подсказка (плейсхолдер).

    Дополнительная информация

  • :default

    Псевдокласс для стилизации элементов, которые используются по умолчанию в группе аналогичных элементов.

    Дополнительная информация

  • :lang

    Псевдокласс :lang выбирает элементы, в зависимости от их языка, который определяется комбинацией атрибута lang, тега <meta> и иногда информацией из протокола передачи данных (такой, как заголовки HTTP).

    Дополнительная информация

  • :root

    Псевдокласс :root нужен для обращения к самому главному родительскому элементу документа. Его также называют корневым элементом. В случае HTML-документа самым главным родителем всей страницы будет тег <html>. Часто :root используется для того, чтобы задавать кастомные свойства.

    Дополнительная информация

  • :indeterminate

    Псевдокласс, который отражает неопределённое состояние чекбокса, радиокнопки или прогресс-бара.
