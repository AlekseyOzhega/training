Анимации

  • CSS-анимации

    Статья об анимации в CSS

    Дополнительная информация

  • @keyframes

    Директива @keyframes используется для создания ключевых кадров CSS-анимаций.

    Дополнительная информация

  • animation-delay

    Свойство задаёт задержку воспроизведения CSS-анимации.

    Дополнительная информация

  • animation-direction

    Свойство сообщает браузеру, должна ли анимация проигрываться в обратном порядке.

    Дополнительная информация

  • animation-duration

    При помощи свойства прописывается длительность одного цикла анимации.

    Дополнительная информация

  • animation-fill-mode

    Свойство сообщает браузеру, нужно ли применять стили ключевых кадров до или после проигрывания анимации.

    Дополнительная информация

  • animation-iteration-count

    При помощи этого свойства можно указать, сколько раз будет проигрываться CSS-анимация.

    Дополнительная информация

  • animation-name

    Чтобы связать ключевые кадры CSS-анимации с конкретным элементом, используется свойство animation-name.

    Дополнительная информация

  • animation-play-state

    Свойство, позволяющее ставить CSS-анимацию на паузу и запускать снова.

    Дополнительная информация

  • animation-timing-function

    При помощи данного свойства можно задать, как будет развиваться анимация между ключевыми кадрами: равномерно, или сначала быстро, потом медленно, или по каким-то сложным внутренним законам.

    Дополнительная информация

  • transition

    Свойство transition используется, когда нам нужно плавно изменить CSS-свойства между двумя состояниями элемента. Например, при наведении мышкой.

    Дополнительная информация

  • transition-delay

    Если нужно плавно изменить какое-то CSS-свойство, но запустить изменение не сразу, а с задержкой, то используем transition-delay.

    Дополнительная информация

  • transition-duration

    Когда нам нужно сделать плавное изменение какого-то CSS-свойства, то браузеру нужно понимать, какое время займёт этот переход. Это время указывается в свойстве transition-duration.

    Дополнительная информация

  • transition-property

    Если нам нужно анимировать переход нескольких свойств элемента (например, цвета и размера шрифта), то удобно перечислить их в свойстве transition-property.

    Дополнительная информация

  • transition-timing-function

    определяет функцию зависимости значения свойства от времени. Другими словами, эта функция определяет скорость течения анимации.

    Дополнительная информация

  • will-change

    Свойство will-change позволяет заранее сообщить браузеру об изменениях (анимация, перемещение и т.д), которые могут произойти с элементом. Так браузер успеет оптимизировать выполнение этих изменений до того, как они произойдут. Это повышает скорость работы сайта.
