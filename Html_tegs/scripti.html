Теги Группы

HTML теги. Скрипты

Тег: <script></script>

Используется для определения сценария на стороне клиента (обычно
JavaScript). Содержит либо текст скрипта, либо указывает на внешний
файл сценария с помощью атрибута src.

Тег: <noscript></noscript>

Определяет секцию, не поддерживающую сценарий (скрипт).

Тег: <template></template>

Используется для объявления фрагментов HTML-кода, которые могут
быть клонированы и вставлены в документ скриптом. Содержимое
элемента не является его дочерним элементом.

Тег: <canvas></canvas>

Холст-контейнер для динамического отображения изображений, та-
ких как простые изображения, диаграммы, графики и т.п. Для рисо-
вания используется скриптовый язык JavaScript.
