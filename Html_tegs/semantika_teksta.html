HTML теги. Семантика текста

  • Тег: <a></a>

    Создаёт используется для вставки ссылок, которые в HTML называются ги-
    перссылками. В качестве гиперссылки можно использовать как текст, так и
    изображение. В браузере гиперссылки отличаются своим видом и цветом.
    По умолчанию HTML ссылки отображаются как подчеркнутый текст сине-
    го цвета.

  • Тег: <em></em>

    используется для выделения текста с особой интонацией. По умолчанию
    такой текст выделяется курсивом, но это настраивается стилями. Стили,
    заданные для тега <em>, могут отличаться в разных браузерах и на раз-
    ных устройствах.

    Дополнительная информация 

  • Тег: <strong></strong>

    используется для выделения важного текста на странице — например,
    чтобы привлечь внимание к тексту или выделить подзаголовок внутри
    абзаца. Обычно такой текст выделяется жирным шрифтом. Не исполь-
    зуйте <strong> для того, чтобы просто сделать текст жирным.

    Пример действия тега

    Дополнительная информация 

  • Тег: <small></small>

    уменьшает размер шрифта на одну условную единицу в сравнении с
    окружающим текстом. Текст можно уменьшить в параграфе <p>, за-
    головках любого уровня или других текстовых блоках. Не используй-
    те <small> для уменьшения размера шрифта у всего блока текста —
    для этого есть CSS-свойство font-size.

    Пример действия тега

    Дополнительная информация 

  • Тег: <s></s>

    используется для определения текста, который больше не актуален.
    Содержимое тега браузеры отображают как перечеркнутый текст.
    Не используйте тег <s> для определения удаленного текста, для
    этих целей используется тег <del>.

  • Тег: <cite></cite>

    (citation source) используется для указания цитаты или ссылки на
    источник информации. Обычно элемент содержит название цити-
    руемого произведения и имя автора или издателя.

    Дополнительная информация 

  • Тег: <q></q>

    (inline quotation) используется для выделения коротких цитат
    (строчная цитата). Как правило, браузеры автоматически устанавли-
    вают открывающиеся и закрывающиеся кавычки в начале и в конце
    выделенного текста.

    Пример действия тега

    Дополнительная информация 

  • Тег: <dfn></dfn>

    (definition - термин, определение) применяется для выделения терминов при
    их первом появлении в тексте. Как правило, в документе, когда упоминается
    новый термин, он выделяется курсивом и даётся его определение. При испо-
    льзовании этого термина в дальнейшем, он считается уже известным читате-
    лю.

    Пример действия тега

    Дополнительная информация 

  • Тег: <abbr></abbr>

    (Abbreviation) определяет аббревиатуру или акроним. Маркировка сокраще-
    ний является полезной информацией для браузеров, систем перевода и поис-
    ковых машин, для пользователя применение этого элемента не видимо.
    Как правило, тег <abbr> применяется совместно с глобальным атрибутом
    title, для отображения полного описания аббревиатуры пользователю при
    наведении.

  • Тег: <ruby></ruby>

    предназначен для добавления небольшой аннотации сверху или снизу от за-
    данного текста. Такая форма записи преимущественно используется для идео-
    графической письменности вроде китайского языка, но может применяться и
    для других языков, если требуется написать один текст над другим.

    Пример 你(nǐ) 好(hǎo)

    Дополнительная информация 

  • Тег: <rb></rb>

    (ruby base) определяет вложенный в него текст как базовый компонент анно-
    тации. Элемент должен располагаться внутри контейнера <ruby> и обычно
    используется совместно с элементами <rt> (формирует аннотацию к базово-
    му тексту, после которого он идет), <rp> (используется для вывода текста в
    браузерах, не поддерживающих элемент <ruby>), <rtc> (поддерживается то-
    лько в Firefox и формирует дополнительную аннотацию к тексту).

    Дополнительная информация 

  • Тег: <rp></rp>

    (reversed parentheses) содержит текст, который будет отображаться в случае,
    если браузер не поддерживает теги <ruby>. Тег <ruby> используется для
    указания произношения слов в японском и китайском языках. Тег <rp> дол-
    жен быть вложен внутрь тега <ruby> и использоваться только совместно с
    ним.

    Дополнительная информация 

  • Тег: <rt></rt>

    (ruby text) Тег <rt> используется для добавления аннотаций сверху или снизу
    от текста, заключенного в тег <ruby>. Тег <ruby> определяет фуриганы (их
    также называют ёмигана или руби), фонетические подсказки в японском, а
    также восточно-азиатских языках.
    Заключенная в тег <rp> аннотация содержит подсказку по произношению
    и переводу основного текста (иероглифов). Аннотация отображается мел-
    ким шрифтом над иероглифами.

    Дополнительная информация 

  • Тег: <rtc></rtc>

    ("ruby text container") обозначает текстовый контейнер внутри <ruby>.
    В основном применяется в качестве описательной части или аннотации
    для иероглифов. Примечание: Тег <rt> используется вместе с тегами
    <ruby> и <rp>: элемент <ruby> состоит из одного или нескольких символов,
    которые нуждаются в объяснении, а необязательный тег <rp> определяет,
    что будет показывать браузер, который не поддерживает элемент <ruby>.

    Пример действия тега

    Китайский иероглиф: 我wǒ 爱ài 学xué 习xí

    Дополнительная информация 

  • Тег: <data></data>

    является новым элементом в HTML5. Он используется для вложения в
    HTML документ данных, таких как денежные величины, величины изме-
    рения и т.д. С тегом <data> используется атрибут value, при помощи кото-
    рого мы сообщаем браузеру, что для указанных данных есть машинный
    аналог, стандартизированная версия, понятная автоматизированным сис-
    темам или скриптам на сайте. К примеру, в тег <data> может быть заклю-
    чен идентификатор (ID) товара.

    Дополнительная информация 

  • Тег: <time></time>

    Элемент служит для разметки дат, времени или периода времени: в содер-
    жимое тега идёт формат для человека, а в атрибут datetime версия для ма-
    шин

    Пример действия тега "12 октября 2021"

  • Тег: <code></code>

    предназначен для отображения одной или нескольких строк текста, который
    представляет собой программный код. Сюда относятся имена переменных,
    ключевые слова, тексты функции и т. д. Браузеры обычно отображают содер-
    жимое контейнера <code> как моноширинный текст уменьшенного размера.

    Пример действия тега

  • Тег: <var></var>

    это фразеологический тег, используемый для определения переменной в мате-
    матическом уравнении или компьютерной программе. Содержимое этого тега
    отображается курсивом в большинстве браузеров. Примечание: Этот тег не яв-
    ляется устаревшим, но с помощью CSS мы можем добиться лучших эффектов.

    Пример действия тега E=mc2

    Дополнительная информация 

  • Тег: <samp></samp>

    (sample - образец) используется для обозначения результата вывода програм-
    мы, скрипта или командной строки. Например, мы можем использовать эле-
    мент <code> для написания листинга программы, а <samp> для результата
    его выполнения. Браузеры отображают его содержимое моноширинным
    шрифтом.

    Пример действия тега

  • Тег: <kbd></kbd>

    используется для обозначения текста, который набирается на клавиатуре или
    для названия клавиш. Браузеры обычно помечают текст в контейнере <kbd>
    моноширинным шрифтом.

    Пример действия тега

    Дополнительная информация 

  • Тег: <sub></sub>

    используется для написания текста ниже базовой линии текста более мелким
    шрифтом. Этот тег может использоваться многими способами, например, мы
    можем обозначать некоторые математические переменные в виде текста с
    нижним индексом.

    Пример действия тега

    Дополнительная информация 

  • Тег: <sup></sup>

    выравнивает элемент как надстрочный благодаря свойству CSS по умолча-
    нию vertical-align со значением super, и по аналогии с элементами <small>
    (определяет текст меньшего размера) и <sub> (определяет текст с нижним
    индексом) устанавливает размер шрифта меньшего размера, чем у родите-
    льского элемента (значение CSS по умолчанию - font-size: smaller).

    Пример действия тега

    Дополнительная информация 

  • Тег: <i></i>

    используется для изменения голоса или настроения части текста, чтобы пере-
    дать его другое качество: официальное название, идиома из другого языка,
    транслитерация или ремарки. Для выделения важного используется <strong>,
    для изменения тона — <em>. По умолчанию браузеры применяют к <i> кур-
    сивное начертание с помощью font-style: italic.

    Пример действия тега

    Дополнительная информация 

  • Тег: <b></b>

    определяет текст, к которому нужно привлечь внимание пользователя, без ка-
    кой-либо дополнительной смысловой нагрузки (в том числе и для поисковых
    систем). В качестве примера текста, который может быть выделен, можно
    привести название продукта, к которому пишется обзор, или недавно введен-
    ный термин, незнакомый еще большинству пользователей, и тд. Содержимое
    элемента <b> браузеры отображают жирным текстом.

    Пример действия тега

    Дополнительная информация 

  • Тег: <u></u>

    В спецификации HTML 4.01 тег <u> использовался для определения подчер-
    кнутого текста. В HTML5 в тег <u> заключается текст, который стилистиче-
    ски отличается от остального текста. Это могут быть слова с орфографичес-
    кими ошибками, слова, которые должны быть подчеркнуты по правилам язы-
    ка (к примеру, в китайском языке имена собственные подчеркиваются) и т.д.
    Содержимое тега <u> в браузерах отображается как подчёркнутый текст.

  • Тег: <mark></mark>

    выделяет части текста, с которыми работает пользователь в данный момент,
    и которые представляют для него важность. Визуально содержимое тега выг-
    лядит как отмеченное маркером.

    Тег <mark> можно использовать, например, для выделения результатов поис-
    ка на странице, либо для того, чтобы подсветить текущую дату в календаре.

    висит от направления текста снаружи. Атрибуты. Скопировать ссылку "Атри-
    буты". Скопировано. К тегу <bdi> можно применить любые глобальные атри-
    буты. По умолчанию у тега есть атрибут dir со значением auto.

  • Тег: <bdo></bdo>

    означает «двунаправленное переопределение» и используется для изменения
    направления текста HTML-документа. Этот тег полезен при работе с языка-
    ми, которые пишутся справа налево: арабским, ивритом или урду.

    Пример действия тега

  • Тег: <span></span>

    строчный элемент, который используется для стилизации части текста. Напри-
    мер, с его помощью можно поменять цвет, размер шрифта или начертание, до-
    бавить и убрать подчёркивание. При этом сам по себе тег не несёт никакого
    смыслового значения.

    Если тег <span> используется без каких-либо стилей или классов, он не вли-
    яет на содержимое.

  • Тег: <del></del>

    используется для выделения текста, который должен быть удален из докумен-
    та, или информации, которая стыла не актуальной. Это может быть полезно
    в определении различий между несколькими версиями документа. По умол-
    чанию, браузеры будут отображать такой текст, как зачёркнутый (перечёрк-
    нутый). Этот элемент обычно используется вместе с тегом <ins> в ситуаци-
    ях, когда часть содержимого заменяется на более актуальную информацию.

    Пример действия тега

    Дополнительная информация 

  • Тег: <ins></ins>

    (insert — вставка) предназначен для выделения текста, который был добав-
    лен в новую версию документа. Подобное форматирование позволяет отс-
    ледить, какие изменения в тексте документа были сделаны. Браузеры обыч-
    но помечают текст в контейнере <ins> как подчёркнутый.
