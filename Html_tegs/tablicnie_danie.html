Теги Группы

HTML теги. Табличные данные

  • Тег: <table></table>

    Элемент для создания таблицы.

    Пример:

    Подпись таблицы
    Заголовок 1 Заголовок 2
    Ячейка 1 Ячейка 2
    Ячейка 3 Ячейка 4
    Нижний колонтитул 1 Нижний колонтитул 2
    Дополнительная информация 

  • Тег: <caption></caption>

    Добавляет подпись к таблице. Вставляется сразу после открывающего
    тега <table>.

    Дополнительная информация 

  • Тег: <colgroup></colgroup>

    Создает структурную группу столбцов, выделяющую множество логически
    однородных ячеек.

    Дополнительная информация 

  • Тег: <col>

    Выбирает для форматирования один или несколько столбцов таблицы,
    не содержащих информацию одного типа.

    Дополнительная информация 

  • Тег: <thead></thead>

    Определяет заголовок таблицы.

    Дополнительная информация 

  • Тег: <tbody></tbody>

    Определяет тело таблицы.

    Дополнительная информация 

  • Тег: <tfoot></tfoot>

    Определяет нижний колонтитул таблицы.

    Дополнительная информация 

  • Тег: <tr></tr>

    Создает строку таблицы.

    Дополнительная информация 

  • Тег: <th></th>

    Создает заголовок ячейки таблицы.

    Дополнительная информация 

  • Тег: <td></td>

    Создает ячейку таблицы.
