HTML теги. Встроенное содержимое

  • Тег: <picture></picture>

    Элемент-контейнер, содержащий один элемент <img> и ноль или несколько элементов <source>.
    Сам по себе ничего не отображает. Дает возможность браузеру выбирать наиболее подходящее
    изображение.

    Дополнительная информация 

  • Тег: <source></source>

    Указывает местоположение и тип альтернативных медиаресурсов для элементов <picture>,
    <video>, <audio>.

    Дополнительная информация 

  • Тег: <img>

    Встраивает изображения в HTML-документ с помощью атрибута src, значением которого явля-
    ется адрес встраиваемого изображения.

    Дополнительная информация 

  • Тег: <iframe></iframe>

    Создает встроенный фрейм, загружая в текущий HTML-документ другой документ.

    Дополнительная информация 

  • Тег: <embed>

    Элемент-контейнер для встраивания внешнего интерактивного контента или плагина.

    Дополнительная информация 

  • Тег: <object></object>

    Контейнер для встраивания мультимедиа (например, аудио, видео, Java-апплеты, ActiveX, PDF и
    Flash). Также можно вставить другую веб-страницу в текущий HTML-документ. Для передачи
    параметров встраиваемого плагина используется элемент <param>.

    Дополнительная информация 

  • Тег: <param>

    Определяет параметры для плагинов, встраиваемых с помощью элемента
