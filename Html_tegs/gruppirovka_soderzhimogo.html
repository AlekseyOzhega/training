Теги Группы

HTML теги. Группировка содержимого

Тег: <p></p>

Параграфы в тексте.

Тег: <address></address>

Задает контактные данные автора/владельца документа или статьи. Отображается в
браузере курсивом.

Пример:

Е-мэйл, почтовый адрес

Тег: <hr>

Горизонтальная линия для тематического разделения параграфов.

Пример:

Тег: <pre></pre>

Выводит текст без форматирования, с сохранением пробелов и переносов текста. Мо-
жет быть использован для отображения компьютерного кода, сообщения электронной
почты и т.д.

Тег: <blockquote></blockquote>

Выделяет текст как цитату, применяется для описания больших цитат.

Пример:

"Примерный текст цитаты"

Тег: <ol></ol>

Упорядоченный нумерованный список. Нумерация может быть числовая или алфавитная.

Пример:

  1. первый пункт
  2. второй пункт
  3. третий пункт

Тег: <ul></ul>

Создает маркированный список.

Пример:

  • первый пункт
  • второй пункт
  • третий пункт

Тег: <li></li>

Элемент маркированного или нумерованного списка.

Тег: <dl></dl>

Элемент-контейнер, внутри которого находятся термин и его описание.

Пример:

Термин
Описание термина

Тег: <dt></dt>

Используется для задания термина.

Тег: <dd></dd>

Используется для описания термина из элемента <dt>.

Тег: <figure></figure>

Самодостаточный элемент-контейнер для такого контента как иллюстрации, диагра-
ммы, фотографии, примеры кода, обычно с подписью.

Пример:

Тег: <figcaption></figcaption>

Заголовок/подпись для элемента <figure>.

Тег: <main></main>

Контейнер для основного уникального содержимого документа. На одной странице
должно быть не более одного элемента <main>.

Тег: <div></div>

Элемент-контейнер для разделов HTML-документа. Используется для группировки
блочных элементов с целью форматирования стилями.
